# Potentiometer

This Arduino code once uploaded, and configured properly and once opening the Serial Monitor (Tools > Serial Monitor), should display the value (from 0 to 1024) of the potentiometer.

![Fritzing Diagram showing a 3 leg potentiometer connected to a breadboard](Potentiometer.jpg)

## Required Hardware:
+ [Arduino UNO R4 WiFi](https://a.co/d/3F1rix2)
+ [Potentiometer](https://a.co/d/dyIsWco)
+ [Breadboard](https://a.co/d/dyIsWco)
+ [Jumper Wires](https://a.co/d/dyIsWco)
